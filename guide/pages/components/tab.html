Tab

Tab in Tab

상위 탭은 component-tab 클래스를 사용합니다.
이 탭의 data-active 속성에 하위 탭 그룹의 data-tab-value 값을 설정하면 해당 탭 그룹이 활성화됩니다.

하위 탭은 각각 data-tab-value 속성 값을 가집니다. 이를 통해 하위 탭과 컨텐츠를 개별적으로 활성화할 수 있습니다.
예를 들어, 상위 탭에 data-tab-value="tab1" , 하위 탭에 data-tab-value="tab1-1" 을 설정하면 해당 탭과 컨텐츠가 활성화됩니다.
  • 구성 요소
    컴포넌트 클래스 : component-tab

Tab Head Sticky

data-props-all-check="true" 속성을 적용하면 Tab Head가 Sticky로 고정됩니다.
  • 구성 요소
    컴포넌트 클래스 : component-tab

Tab Head Sticky : 위치 조절

data-props-sticky 속성 값에는 고정할 요소의 클래스를 지정해야 하며, 적용 시 해당 요소의 높이만큼 탭 위치가 조정됩니다.
data-props-sticky=".height-controll" 속성을 적용하면 height-controll 요소의 높이만큼 탭 위치가 조정됩니다.
  • 컴포넌트 클래스 : .component-tab
  • 컴포넌트 자식 클래스 : .tab-head, .tab-body, .tab-label, .tab-content
  • Tab Head Sticky : data-props-sticky="class" - class의 높이에 따라 Tab 위치를 조절할수있다.

Tab Scroll

component-tab 에 tab-scroll 클래스를 추가하면 탭 클릭 시 좌측으로 이동하며 가로 스크롤이 활성화됩니다.
